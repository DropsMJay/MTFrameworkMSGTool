Python scripts for extracting/inserting text data from CAPCOM's MT Framework engine.

For now, its only working on Resident Evil 5 (2009)

How to use:

python [ScriptGoesHere] [FolderPathforFilesHere]

Example:
Extract: python REMGSExtractor.py C:\ResidentEvil5TextData<br/>
Insert: python RE5MGSInseter.py C:\ResidentEvil5TextData

Dependencies:
Python 3.14
